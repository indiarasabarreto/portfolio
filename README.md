````
## 📌 O que você vai construir

        Um script que:

            - Organiza arquivos por tipo (PDF, imagem, etc.)
            - Move automaticamente para pastas

        🛠️ Tecnologias:
            - Python
            - os, shutil


        🧩 Passo a passo
            1. Ler arquivos de uma pasta
            2. Identificar extensão (.pdf, .jpg…)
            3. Criar pastas automaticamente
            4. Mover arquivos

        💡 Extra (diferencial)
            - Rodar automaticamente (cron ou loop)





````

# 📊 PROJETO 2 — Análise de Dados

## 📌 O que você vai construir

````
Análise de vendas (ou dados públicos)

🛠️ Tecnologias:
    - pandas
    - matplotlib

🧩 Passo a passo:

1. Baixar dataset (Kaggle ou fake)
2. Limpar dados
3. Fazer perguntas:
    - Produto mais vendido?
    - Mês com mais vendas?
4. Criar gráficos

💡 Extra
    - Usar Jupyter Notebook 
````
# 🌐 PROJETO 3 — API Backend (ESSENCIAL)

## 📌 O que você vai construir

````
Uma API de tarefas (to-do list)

🛠️ Tecnologias:
    - FastAPI (recomendado)
ou
    - Flask

🧩 Funcionalidades:
    - Criar tarefa
    - Listar tarefas
    - Atualizar
    - Deletar

🧩 Passo a passo:
    1.Criar projeto
    2.Criar rotas (GET, POST, PUT, DELETE)
    3.Usar lista ou banco simples (SQLite)
    4.Testar com Postman

💡 Extra (MUITO forte)
    - Deploy (Render ou Railway)
````
